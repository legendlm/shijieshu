# 世界书
世界书 是大连浮点映射软件科技有限公司开发的开放式小说APP，核心是让用户通过自由文字指令，与 AI 共创剧情、角色扮演、搭建世界观，主打高自由度文字冒险。
核心定位：把 “被动看小说” 变成 “主动玩故事、一起写故事”。

## 官方网站：https://www.shijieshu.cc

## 简介
一句话，改写命运；一个念头，重塑世界。
开放式小说，不再被动阅读，不再等待更新。
这是你亲自掌舵的虚拟世界。
AI实时推演，角色任你号令，世界随心而变。

## 特性
智能对话：与AI进行自然流畅的对话互动，实时创作您满意的故事。
动态叙事：故事走向由您决定，每一次选择都影响剧情发展，创造独一无二的阅读体验。
个性世界：定制专属角色、场景和世界观，打造完全属于您的故事宇宙。
开放生态：提供另存和魔改功能，仿照大神快速上手开启创作之旅。
社区分享：分享您的创作到市场，发现更多志同道合的故事爱好者。
隐私安全：本地存储您的创作数据，所有对话记录严格加密保护，只属于您自己。

## 核心功能
### 自由剧情推演
输入任意指令（如 “探索遗迹”，“和她聊天”），AI实时生成剧情、场景、对话，无固定剧本。
支持自定义世界规则、世界元素, AI不只是角色扮演，而是连续稳定的推演虚拟世界，最终以主角的第二人称视角进行展现。
记忆机制：本地记忆库自动总结、关联、记忆，对话上千轮不失忆不混乱。

### 世界观设定系统
世界规则：设定全局基础规则，奠定整个虚拟世界的基础
世界元素：分为角色、场景、事件、道具，AI按设定的智能、触发、随机3种方式管理，动态注入提示词和元素详情设定。

## 核心AI技术
### 大语言模型（LLM）— 核心引擎
基于先进大语言模型负责：
自然语言理解（NLU）：解析用户自由指令、意图、情绪。
自然语言生成（NLG）：实时生成连贯剧情、对话、场景描写，匹配指定文风（严肃 / 轻松 / 古风等）。
多轮对话管理：维持角色人设一致性、剧情连贯性，支持长会话记忆。

### 智能记忆与检索技术
长期记忆存储：向量数据库存储世界观、角色设定、历史剧情，避免 “失忆”。
智能检索触发：语义匹配用户输入与世界书设定，动态注入背景信息。
上下文感知：实时关联前文剧情、角色关系、世界规则，生成符合逻辑的后续内容。

### 个性化生成与风格迁移
风格自适应：AI 识别用户选择的题材（玄幻 / 言情 / 科幻），自动切换文风、世界观设定、角色行为逻辑。
人设一致性强化：基于角色卡（性格、背景、关系），AI 生成符合人设的对话与行为，避免 OOC（Out of Character）。
偏好学习：根据用户历史指令、选择、反馈，逐步适配用户偏好的剧情节奏、风格、内容方向。
